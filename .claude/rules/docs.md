# Documentation Rules

When editing files under docs/:

- anthropic-source-map.md must classify every capability as Native, Convention,
  or Enforcement
- upstream/doc-index.md must have verification dates for all Anthropic doc pages
- Never cite community repos or third-party sources as authoritative
- Architecture decisions go in .kanban/decisions.md (DECISION-NNN format)
- Keep documentation concise and actionable
