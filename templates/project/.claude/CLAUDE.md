## StoryForge Integration

This project uses StoryForge for delivery tracking.
- Kanban board: `.kanban/board.md`
- Active stories: `.kanban/stories/`
- Backlog: `.kanban/backlog.md`

Follow the StoryForge operating model:
1. Check the board before starting work
2. Work within the active Story scope
3. Update artifacts when work completes
